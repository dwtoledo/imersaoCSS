In first section, I used the grid display CSS properties to create 3 columns (Area 1, Area 2 and Area 3). Each column has a 1/3 of screen width, 3/5 of screen height and different colors.


Inside the Area 3, I added an internal area (in white) and used absolute & relative position properties to align it exactly in the center (X-axis) and middle (Y-axis). The code was written to be flexible, so if the screen size changes, there is no need to adjust the absolute & relative values.


The same position properties type was used to align Area 1, 2, 3, 4 & 5 texts inside their respective areas.

In second section I used the float, overflow and clear CSS properties to move Area 4 to the right, Area 3 to the left and this Text below the Areas 4 & 5. Details:


float: "right" for Area 4;

float: "left" for Area 5;

overflow: "hidden" on the parent aiming to adjust the vertical height;

clear: "both" to identify if there are float elements on the left or right:

if YES (this case), the Text element is positioned below the Areas 4 & 5;

if NO, the Text element keep its position.
